# Prediction-and-Weather-Driven-Analysis-of-Declining-Bird-Species-in-Karnataka
Machine learning framework for retrospective classification of bird population trends in Karnataka using eBird and climate data (2018–2024).

## Data

Raw eBird data are not included due to licensing restrictions.

They can be accessed from:
https://ebird.org/data/download

Processed data used in this study are available via Zenodo.
